# 📍 Why Emotions Matter

1. Attention, Memory, and Learning
2. Decision Making
    - 情緒會影響決定。養成習慣檢查自己的情緒（問自己「你現在感覺怎麼樣？」）
3. Relationships
    - 想想自己對某個人的情緒，是不是影響我跟他怎麼互動。
4. Physical and Mental Health
5. Creativity
